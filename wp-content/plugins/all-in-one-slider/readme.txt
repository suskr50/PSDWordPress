All in One Slider is an amazingly easy to use responsive image slider plugin for your WordPress site.
 
== Description ==
All in One Slider is an amazingly easy to use responsive image slider plugin for your WordPress site. It brought 2 very well known jQuery sliders in same platform and you can switch to another slider from one slider just in single click.

= Included Sliders =
*	Bootstrap Carousel
*	Camera Slider
*	and many more are coming with next update ...

= How to use =
[youtube https://www.youtube.com/watch?v=w7WvfigfR28]

== Installation ==

1. Upload `all-in-one-slider` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
